# Nobox server

## Requirements
- Docker ( >20.10.13 )
- Node ( > 16.13.0 )

## Usage
Run the following Commands
- `npm install`
- `npm run setup`
- `npm run start:debug`

## To Clear Mongo Data
- `npm run setup -- --rebuild`
# Notes
- You have to enableDeveloperMode for each recordSpace to get list of endpoints;
