### Requirements
- Docker ( >20.10.13 )
- Node ( > 16.13.0 )

### Local Installation

Follow these steps:
- Install And Start Docker if you haven't
- Run: `npm run setup:mongo-replica`
- Run: `npm install`
- Run: `npm run dev`


### Non-Local Installation

Follow these steps:
- Request for the `env/.dev.env` and paste it in
- Run: `npm install`
- Run: `npm run start`
